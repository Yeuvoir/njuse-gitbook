### 实践项目

需求课有和其他企业产品部门的合作项目（我们这届是和 vivo 公司）。匡匡一般在学期初的序言课时会说明报名申请的事项。

实践项目主要任务是为企业设计功能构思产品，最后会去企业做汇报，十分有趣。

本项目可以**替代大作业**，推荐大家报名参加。
