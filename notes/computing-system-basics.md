# 计算系统基础

南软另一门大一课程，如果说 CPL 是入学的当头一棒，那么计基就是未来的习以为常。

计算系统基础在每个学年的上学期和下学期都会开设。上学期的课程主要面向软院的大一同学，使用 DLX 作为课程中的汇编语言。下学期的课程主要面向智软和有意向转专业的大一同学，由毛云龙老师执教，使用 RISC-V 作为课程中汇编语言，**风评很好**。

{% hint style="danger" %}
软院的孩子们别想着缓修计基，操作难度比较大。
{% endhint %}
